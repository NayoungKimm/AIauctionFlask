About Villa Space

Introduce our service

이 웹은 성신여자 대학교의 교내 대회에서 진행한 프로젝트입니다.

저희를 소개할게요.

저희는 성신여자 대학교 AI학과 21학번 친구들 4명이 모여 이 프로젝트를 진행하였습니다.

빌라 관련 데이터를 모으고, 데이터를 분석하여 알고리즘을 만든 후 그에 맞는 분류 모델을 적용시겨 부동산의 가격을 예측해주는 서비스를 만들었습니다.

위 서비스는 빅데이터를 바탕으로 분석 후 , 다양한 변수를 고려하며 예측 가격을 제공해 줌으로써 높은 정확도를 기대해 볼 수 있으며,

간단하고 편리한 UI로 사용자에게 쉬운 접근성을 제공해 줍니다.

적용 기술

Villa Space를 만들기 위해 적용한 기술을 소개합니다.

visual studio code

visual studio code를 이용해 프론트 작업을 하였습니다.

Pycharm

Pycharm을 활용해 데이터를 분석하고 알고리즘을 개발하였습니다.

머신러닝

시계열 데이터를 이용하여 반복 기계학습을 통한 결과 추론을 한다.

GitHyb Desktop

깃허브 데스크탑을 통해 협업 프로젝트를 진행하였다. 작업한 코드를 올리고 받아오며 팀원들과 상호 작용하며 프로젝트를 진행하였다.
